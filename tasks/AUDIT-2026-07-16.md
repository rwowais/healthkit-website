## REL-9 (fetch timeout) — SHIPPED 2026-07-16 (commit e3d40da)
Injected a 10s AbortController timeout via the Supabase client's `global.fetch`
(new `src/lib/fetchWithTimeout.ts`) so a black-holed connection can't hang
`load()` forever (infinite skeleton / stuck post-sign-in nav). A timed-out
request surfaces as an error → existing offline fallback (local cache) applies.
Gates: tsc clean · **full suite 627 passed / 18 skipped** · build compiled ·
4 new unit tests. Not browser-observable (needs a stalled connection); logic
unit-tested + load()'s error path already covered.
